### 一：客户端（frontend）
整个项目采用前后端分离架构，前端采用vue编写

### 二：Api网关 （funnel）
客户端访问后端服务时，有一个统一的入口，而且可以做统一的权限校验，该网关是我用java写的一个简单的网关应用
登陆权限部分采用的jwt技术实现
